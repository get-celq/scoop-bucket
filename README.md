## Install celq using Scoop

This repository contains the configuration to install `celq` on Windows

```powershell
scoop bucket add get-celq https://github.com/get-celq/scoop-bucket
scoop install get-celq/celq
```

See [Scoop's README](https://github.com/ScoopInstaller/Scoop) for instructions on how to install Scoop on Windows.

### Uninstalling

Run:
```powershell
scoop uninstall get-celq/celq
scoop bucket rm get-celq
```
